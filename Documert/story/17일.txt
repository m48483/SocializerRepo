# 17일 일과 시작
플레이어: 문밖에 일정한 구두 소리가 점점 가까워졌다.\ 이윽고 문이 열리며 단정한 차림의 여성이 한 손에 고급 쇼핑백을 들고 들어왔다.
//
 프랜시스 무어, 내가 기다리던 그 브로커였다.\ 한참을 잠적해 있다가 이 시기에 다시 활동을 재개한 이유가 무엇일까. 부? 명예?
//
 그녀를 보며 다시 머릿속으로 되새겼다. 뭐든 확실한 것이 좋으니까.\ 어머니가 벤야민의 죽음을 덮은 것은 다 뜻이 있을 것이다. 그것에 있어 이 여자는 위험요소다.
//
그녀가 뭔가 알고 있는 낌새를 보이면 수단과 방법을 가리지 않고 입을 막으면 되고 아무것도 모른다면 대충 경고만 하고 보내면 된다. 
//
 천천히 다가온 그녀는 올리비아에게 이미 나에 대한 이야기를 들었는지 살짝 긴장을 한 채 내 앞에 앉았다.
//
 나는 자연스럽게 내 소개를 하며 악수를 청했다.\ 고상한 태도와는 다르게 그녀의 손은 투박하고 거칠었다.

프랜시스: 제가 죽기 전에 시메 영웅의 따님을 만나 뵙게 된다니. \좋은 일로 뵙는 것이면 더할 나위 없는 영광이었을 텐데..

플레이어: 애교를 띤 미소와 살짝 주눅 든 듯한 목소리는 상류층이 좋아할 만하였다.\ 본인을 낮추는 태도와 적당히 보기 좋은 얼굴, 이런 것들이 그들의 셀링 포인트이기도 했다.

프랜시스: 아차, (플레이어 이름) 리히터씨… 제가 어떻게 부르면 될까요?

플레이어: 그냥 (플레이어 이름)으로 부르십시오.
//
 어머니는 내가 리히터로 불리는 것을 싫어하셨다.\ 리히터 가문에서 성으로 불릴 수 있는 건 어머니와 진짜 (플레이어 이름)뿐이었다.

프랜시스: ...올리비아님 건으로 부르신 것... 맞죠? 시국이 시국인지라 제가 더 조심했어야 했는데…\ 워낙 그분이 자유분방하시기도 하고, 그동안의 신뢰도 있어서…

플레이어: 계층 상승을 원하는 자들은 정보원들에게 납작 엎드려 본인의 무해함을 증명하고는 했다.\ 현재 시메에서 정보부 눈 밖에 들면 할 수 있는 일이 없으니 당연한 행위이긴 했다.
//
그들은 우리가 별다른 행동을 취하지 않아도 노예를 자청했다.\ 이것이 그들의 생존 방식이었다. 그리고 이것은 내 생존방식이기도 했다.
//
내 눈에 들기 위해 어리숙한 척하는 그녀의 모습에 조금은 동정심이 일었다.

프랜시스: 얼마전에 피살 사건...도 있었으니까… 당분간은 정말로 조용히 있으려고 했어요.\ 앞으로는 (플레이어 이름)님 심기에 거슬리는 일은 없도록 할게요.

플레이어: 그녀는 벤야민의 일을 언급하고 내 심기를 불편하게 했을까 눈치를 봤다.
//
 피살 사건 이후 공백기에 자택이 아니라 호텔에서 지내셨더군요.\ 신트라 호텔은 외국인 투숙객이 많아 안전한 선택지라 생각되지는 않습니다만.

프랜시스: 고급 호텔은 저 같은 사람에겐 부담이 꽤 커요. 그리고 그때는 그저 몸을 숨겨야겠다는 생각밖에 없었어요.
//
 이번 일과는 상관없을 수 있지만… 다음이 제 차례일 수도 있겠다 생각했어요.\ 저희 사이에서 경쟁자를 제거하는 일은 비일비재하니까요.
//
어떻게든 살아남아서 누릴 거 누리면서 사는 게 제 목표에요.\ 얇고 길게.
//
그녀가 싱겁게 웃었다.
//
지금은 별다른 위협이 없어서 다시 그레이타운에서 지내고 있어요.\ 혹여 제가 의심된다면 조사해보셔도 괜찮아요. 정말 방에서만 지냈거든요.

플레이어: 실제로 그녀는 호텔 방 밖으로 나오지 않았고 마주친 이도 직원밖에 없었다.\ 그녀가 말한 목표와 들어맞는 행동이었다.
//
그런 의도로 물어본 것이 아니니 안심하십시오.
//
그 뒤로 이어진 시시콜콜한 대화는 별 소득이 없었다. 
가도 된다는 나의 말에 그녀는 이것이 본 목적인 듯 손에 들고 있던 쇼핑백을 내게 들이밀었다. 

프랜시스: 약소하지만, 사죄와 위로의 마음을 담아 준비한 제 마음이에요. 아, 어떠한 의도도 없으니 편히 받아주세요.

플레이어: 필요 없습니다. 더더욱 내가 당신에게 위로를 받을 만한 위치의 사람도 아닙니다.
//
본인의 행동이 고위층에게 불쾌할 수 있음을 분명 알고 있음에도 그녀는 제 물건을 기어이 책상위에 올려놓고는 자리에서 일어섰다.

프랜시스: 분명 (플레이어 이름)님 마음에 드실 거예요.

플레이어: 그녀가 나를 보며 씽긋 웃었다. 나는 한숨을 쉬며 그녀를 따라 문까지 배웅했다.
//
오늘 같은 행동은 하지 마십시오. 불쾌합니다.
//
그리고 이제 돈을 꽤 모으셨을 텐데 신분을 바꾸는 것이 좋을 겁니다. 본인 말처럼 오래 살아서 누리려면.
//
시메에서 밀수는 중죄입니다. 밀수업자의 말로는 당신이 제일 잘 알 것이고. 저희가 언제까지고 눈 감고 있을 것이라 자만하지 마십시오.
//
자리로 돌아가기 위해 몸을 돌리자 내 쪽을 계속 힐끔 보고 있던 딜런과 눈이 마주쳤다. 
//
그는 멋쩍은 웃음을 보이다 이내 본인의 일에 관심을 돌렸다. 저 가방 안에 든 것이 뭔지 궁금해 죽겠다는 표정으로.

카드만 챙기고 나머지 내용물은 쓰레기통에 버렸다. 얼마 지나지 않아 딜런이 들어와 내 안부를 살폈다. 그는 어지간히 모욕적이었는지 여전히 상기되어 있었다.

그의 어깨를 살짝 토닥이며 목소리를 낮춰 말했다.

딜런, 이건 좋은 기회가 될 수도 있어요. 내가 따로 조사해볼 테니… 오늘 일은 우리 둘만 알고 있는 걸로 합시다.
