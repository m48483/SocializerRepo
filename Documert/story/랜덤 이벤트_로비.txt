# 랜덤 이벤트_1 (로비)
플레이어: 휴게실 근처를 지나자 삼삼오오 모여 대화를 나누는 소리가 들렸다.
직원A: 그래서 어떤 라인 탈 건지 정했어? \우리 같은 말단들이 오래 살아남으려면 이거 밖에 없어.
직원B: 저… 들어온지 얼마되지도 않았는데… \이렇게 빨리 정해야 하나요? \사실 뵌 적도 없는 분들이라...
직원C: 이런 건 확실히 할수록 좋은 거야. \괜히 간 보다가 박쥐 놈들처럼 되는 거야. \실적도 못 쌓고 승진도 밀리고 평생 쌩고생만 하다 인생 종치는 거지 뭐.
직원A: 전에도 말했다시피 (플레이어 이름)님이 제일 무난하고 안전해. \지금까지 해오셨던 것도 있고 배경도 그렇고… \부장이 안된다고 해도 뭐 건들 수 있겠어? //
솔직히 벤야민 그 사람이 친 사고 수습하느라 좀 불안하긴 했지만 이젠 걸림돌도 사라졌으니… \그 뭐야 교정소에서 온 땅개 새끼도 기회를 놓칠 새라 알랑거리더만.
직원B: 벤야민…? 혹시 얼마 전에 돌아가셨다는… \그렇게 말해도 되는 거예요?
직원C: 야, 여기에 그 놈 좋아하는 사람 단 한 명도 없어. \(플레이어 이름)님도 그러고 있는 게 자기 남편이니 어쩔 수 없이 챙겨 주신 거지.
//
차라리 일찍 뒤진게 부장님한테도 (플레이어 이름)님한테도 잘 된 일이야. \목적은 이미 다 이뤘겠다 따로 손쓸 필요도 없이 알아서 퇴장해줬으니.
//
플레이어: 솔직하고 신랄한 대화에 픽 웃음이 났다. \그제야 날 인식한 그들은 깍듯이 고개를 숙여 인사했다. \바들바들 떨리는 몸이 그들이 꽤 놀랐다는 걸 증명해주었다.
//
함부로 입을 놀린 책임은 물어야겠지만 구태여 내 편인 사람들을 적으로 돌릴 생각은 없다. \저들이 미래를 위한 거름이 되어줄 테니.
//
재밌네요. \근데 다들 교육 좀 다시 받아야겠어. \우리는 죽어서도 입만 가라앉아야 하는데… 그죠?
//
내 농담에 다들 어색하게 웃었다.
//
세 치 혀가 사람 잡는다는 말도 있잖아요. \내가 아니라 다른 사람이면 어쩔려고 그랬어요? \이번 한 번만 봐드리죠. 
//
계속 지켜볼 거예요. \들어가봐요.
//
그들은 부리나케 휴게실을 떠났다. \다들 너무 풀어졌다.

직원B: 아버지 쪽을 닮았나보지…
//
아, 이거 얼마전에 들은 건데 방계에서 들였다는 소문이 있더라.

동생도 사고로 죽어, 남편은 정신병원에서 죽어, 사위도 칼 맞아서 죽어… \진짜 보통 정신력으론 버틸 수가 없네.
