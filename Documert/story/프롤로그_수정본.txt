# 파트 1 (검정 화면 사용)

플레이어: 그날은 여느 때와 다를 바가 없는 평범한 날이었다. \새로운 근무지 에버트는 평소와 같이 희뿌연 안개가 가득했고 바다에서 오는 눅눅하고 무거운 공기는 나를 잡고 끌어내리려는 듯 내 몸과 뒤엉켰다.
//
찝찝한 공기와 함께 늘어지는 걸음으로 출근하고, 사람들을 모니터링하고, 서류에 파묻혀 일하다 퇴근을 하고, 벤야민이 아이와 함께 노는 모습을 보며 작은 안식을 얻는 그런 일상의 연속일 줄 알았다. \내 남편인 벤야민이 난도질당한 시체로 발견되기 전까진.
//
그는 불법 중개상들을 단속을 위해 에버트의 3계층 시민들이 사는 빈민촌 그레이타운으로 향했다. \악명높은 그레이타운이지만 단속은 도시와 근접한 외각에서 이뤄지고 형식상 하는 것이라 위험한 일이 아니었다. \그가 맡은 부분은 예술품이었기 때문에 더더욱.
//
그는 애초에 정보부과 어울리지 않은 사람이라 내가 위험한 일을 배정하지 않았으니. \그를 정보부에 오래 둘 생각도 아니었고.
//
그는 그레이타운 외각이 아닌 중심지와 가까운 곳에서 발견되었다. \대체 왜 나의 통제를 벗어난 걸까.
//
현장은 오물 냄새와 비릿한 피 냄새가 섞여 속이 뒤집힐 것 같이 역겨웠다. \이곳은 변한 것이 없었다. \정보부 부장인 어머니가 사람들을 통솔하며 그곳을 수습하고 있었다.
//
마침내 마주한 벤야민의 모습은 현실이 더 지독하다는 것을 알려주려는 듯 상상보다 끔찍했다. \그의 얼굴은 도저히 알아보기 어려울 정도로 자상이 가득했다. \날이 잘 선 흉기를 사용한 모양이었다.
//
체형, 그가 입었던 옷 -피로 검붉게 물들어져 있었다-, 늘 하고 다니던 시계 그리고 그의 손에도 내 손에도 껴있는 반지는 그 덩어리가 벤야민임을 분명히 시사했다.
//
그에게 원한이라도 있는 사람이 한 것일까? \정보부에서 그의 평판이 좋지는 않았지만 규정을 어기지 않은 동료를 개인적으로 살해한다는 것은 있을 수 없는 일이었다. \정보부를 제외한다면 그의 인상은 좋은 편이었다. 어수룩하여 속된 말로 호구에 다름없었으니까.
//
그렇다면 그에 대한 원한이 아니라 나와 어머니, 정보부에 대한 경고? 그렇다면 다음은?
//
얼마 지나지 않아 현장 주변에서 흉기를 쥔 채 피를 흘리고 쓰러져 있는 3계층 청년이 발견되었다. \용의자인 그자가 소지하고 있던 흉기는 날이 잔뜩 무뎌 있어 이런 정교한 상처를 내기엔 무리가 있었다. \용의자는 이송 중 과다출혈로 사망했다.

// 
벤야민의 죽음은 그의 금품을 노린 3계층 강도의 살인으로 처리되었다. \미심쩍은 부분이 많았지만, 어머니는 사건이 커지기를 바라지 않았다. \그의 장례는 간단히 이루어졌다.
//
한 줌의 재가 된 그의 죽음을 추모하는 자리에 온 사람은 많았으나 진심으로 슬퍼하는 이는 몇 안 되었다. \그의 가족들과 우리 아이 헤더 정도. \나도 그를 위해 눈물 한 방울 흘리지 않았으니 그런 사람들을 뭐라할 자격은 없었다.
//
그런 나를 보고 그의 부모는 이성을 잃고 악담을 퍼부었고 -그러나 그 발악마저 곧 제지당했다.- 나를 지지하는 직원들은 차기 부장에 걸맞은 평정심이라 평했으며 어머니는 자신의 가르침을 수행하는 피조물에 만족스러운 미소를 보냈다.
//
그의 죽음에 동요하는 일은 나를 선택한 자비롭고 두려운 나의 구원자, 어머니의 뜻을 거스르는 행동이었다. \여기서 더 궁색한 변명을 해보자면 나의 아이를 위해서도 나는 약한 모습을 보여선 안 되었다.
//
그를 죽인 목적이 무엇인지 모르는 상황에서 동요하는 모습을 보였다가는 다음 표적이 헤더가 될 수도 있다.
//
이제껏 신은 나의 작은 안식들을 모두 앗아갔다. \내게 자신의 이름, 지위, 삶 모든 것을 빼앗겨놓고 어린 날 위로해주던 불쌍하고 순수한 영혼 (플레이어 이름)도 그러했고, 따뜻하고 보드라운 바람처럼 날 포용해주던 벤야민도.
//
본래 3계층 출신인 나는 그런 사치를 누릴 자격이 없다는 듯이. \혹여 우리 눈에 띌까 잔뜩 웅크려 불편한 자세로 고개를 푹 숙인 채 이곳을 청소하고 있는 저 사람들처럼 내 주제에 맞게 땅바닥에 기어서 다니라는 듯이.
//
내 손을 잡고 있던 작은 손을 꽉 쥐자 따뜻한 온기가 느껴졌다. \나는 다시는 내 것을 빼앗기지 않을 것이다. \날카로운 가시들이 내 몸을 찢고 관통해 피투성이가 되더라도 지켜낼 것이다.
//
그러곤 이 세계의 가장 높은 곳에 올라가 잘난 신의 낯짝에 침을 뱉어 주리라.
//
온갖 감정들의 덩어리가 나의 목을 꾸욱 눌러 숨이 막혀왔다.

*로비 배경 사용
//
정보부는 빠르게 그의 모든 흔적을 정리했다. \그의 물건은-그날의 소지품을 포함해서- 내게 돌아왔고 문서는 어머니의 집무실에 딸린 문서고로 들어가거나 폐기되었다.
//
그는 정보부에 존재하지 않았던 사람이 된 것이다.
//
멍하니 물건들을 정리하다 그가 늘 들고 다니던 공책이 없음을 깨달았다. \그는 늘 공책을 들고 다니며 평화로운 모습들을 스케치하곤 했다.
//
현장에서도 없었나? \아니, 폐기된 거겠지. \과거의 평안을 잠시나마 떠올릴 수 있는 존재가 없다는 사실에 조금은 아쉬운 기분이 들었다.

*집 배경 '집_호감도 변경 수정본_1_1' 사용
//
집에서는 아이의 작은 훌쩍임과 술잔을 내려놓으면서 나는 날카로운 소리 외에는 정적만이 흘렀다. \정적을 깨고자 텔레비전을 켜면 선량한 1계층 청년이 그레이타운에서 자선활동을 하다 3계층 강도에게 살해당했다는 보도가 이어졌다.
//
그를 향한 추모가 이어졌고 3계층 시민에 대한 여론은 더욱 악화되었다. \계층 간 차별이 공고해지길 원하는 정보부는 크게 힘을 안 들이고 이득을 얻은 셈이었다.
//
그런 생각을 하면 왜인지 모르게 속이 울렁거려 모든 것을 게워냈다. \술을 많이 마신 탓일까.
//
공허, 불안, 무력감 등의 감정이 잉크가 천천히 퍼지는 것처럼 내 몸을 점점 검게 물들여 갔다.
//
규정보다 빠르게 복귀신청을 했다. \그냥 뭐라도 해야 살 거 같았다.
